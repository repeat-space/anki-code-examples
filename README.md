# Anki Code Examples

## Usage

```sh
$ python3 src/tests/1_get_deck_names_test.py
$ python3 src/tests/2_get_cards_from_deck_test.py
```

## Install

```sh
$ pipenv install --dev
```

## Tests

```sh
$ ./scripts/test.sh
```

## Formatting

```sh
$ pipenv run autopep8 --in-place src/**/*.py
```

## Tips

### SQLite

```sh
$ sqlite3 data/profile/collection.anki2
```
